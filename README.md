# Solar System Simulation

A minimalist 3D solar system simulation built with Three.js. Explore the planets in your browser with smooth interactions and clean design.

## Features

- **Interactive 3D View** - Rotate, zoom, and explore
- **Planetary Information** - Click planets for details
- **Adjustable Speed** - Control simulation speed
- **Toggle Elements** - Show/hide orbits and labels
- **Responsive** - Works on desktop and mobile

Controls

    Mouse Drag: Rotate camera

    Mouse Wheel: Zoom in/out

    Click Planets: View information

    Toggle Orbits: Show/hide orbital paths

    Toggle Labels: Show/hide names

    Reset View: Return to default position

Technical Details

Built with:

    Three.js for 3D rendering

    HTML5 Canvas

    CSS3 animations

    Vanilla JavaScript
